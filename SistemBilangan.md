2. Konversikan bilangan desimal di bawah ini ke dalam bilangan biner
   
      - 1234
  
            1234:2=617, sisa 0
            617:2=308, sisa 1
            308:2=154, sisa 0
            154:2=77, sisa 0
            77:2=38, sisa 1
            38:2=19, sisa 0
            19:2=9, sisa 1
            9:2=4, sisa 1
            4:2=2, sisa 0
            2:2=1, sisa 0
            1:2=0, sisa 1
